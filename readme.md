# 🎓 Student Performance Prediction using Linear Regression

## 📌 Project Overview
This project predicts a student's exam score based on various academic factors such as study habits, attendance, and previous performance using **Linear Regression**.

The goal is to understand how different factors influence student performance and to build a simple yet effective predictive model.

---

## 📊 Dataset
The dataset includes features such as:
- Hours Studied  
- Attendance  
- Previous Scores  
- Access to Resources  

**Target Variable:**
- Exam Score  

---

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  

---

## ⚙️ Machine Learning Workflow

1. Data Loading  
2. Data Cleaning (handling missing values & duplicates)  
3. Feature Selection  
4. Encoding categorical variables  
5. Feature Scaling (Standardization)  
6. Model Training (Linear Regression)  
7. Model Evaluation  
8. Regularization (Ridge Regression)  

---

## 🤖 Models Used

### 🔹 Linear Regression
- Baseline model used for prediction  
- Simple and interpretable  

### 🔹 Ridge Regression
- Used to reduce overfitting  
- Adds penalty to large coefficients  

---

## 📈 Model Performance

- **R² Score:** ~0.70  
- **MSE:** (varies based on dataset split)  
- **RMSE:** (approximate prediction error in marks)  

---

## 🧠 Key Learnings

- Importance of data preprocessing  
- Feature selection improves model performance  
- Scaling features is essential for regression models  
- Regularization helps prevent overfitting  
- Model performance depends heavily on data quality
